数据操作

发表于 2019-02-25

插入数据

  • insert into 表名 values(值1,值2,…),（值1,值2,…）
  • insert into 表名(字段1,字段2,…) values(值1,值2…),(值1,值2,…)
    值

数字、“字符串”、时间

时间函数
  • 年月日时分秒
    now();
  • 年月日
    date(now())

  • 年、月、日
    year(now());
    month(now());
    day(now())

    删除数据

  • 删除表中所有数据
    delete from 表名；

  • 删除满足条件的记录
    delete from 表名 where 条件语句；

    修改

  • 修改该字段所有的值
    update 表名 set 字段名=新值；

  • 修改满足条件的该字段的值
    update 表名 set 字段名1=新值，字段名2=新值 where 条件语句；

    获取数据

  • 获取所有的数据
    select * from 表名;

  • 获取指定列的数据
    select 字段名1，字段名2 from 表名；
  • 获取指定条件的列的数据
    select 字段名1，字段名2 from 表名 where 条件语句；
  • 自定义列名显示
    select 字段名1 as 新字段名1，字段2 from 表名 where 条件语句；
  • 对查询结果重新赋值（一般针对布尔数据类型）
    select if(字段名，值1，值2) as 新字段名 from 表名
    select case 字段名 when 1 then ‘值1’ else ‘值2’ end from 表名
    select if(字段名相关的条件语句，值1，值2) as 新字段名 from 表名
  • 查询且以指定符号合并（bit类型合并无效）
    select concat(字段1，’:’，字段2) as 新字段名 from 表名
  • 模糊查询
    利用模糊条件来查询
  • 排序
    select from 表名 order by 字段名 asc默认升序(desc降序)；
    次级排序
    select     from 表名 order by 字段名1 desc，字段名2 desc;
  • 限制指定数量条记录
    select from 表名 limit 数字；
    跳过指定数量的记录后查询指定数量的记录
    select     from 表名 limit 数字 offset 数字；

条件语句

  • 判断相等
    =
  • 判断不相等
    <>
  • 模糊条件
    like ‘小%_’;
  • 零或多个任意字符
    %
  • 一个任意字符
    _
  • and 、or 、not
