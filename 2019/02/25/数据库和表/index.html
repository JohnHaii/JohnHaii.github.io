数据库和表

发表于 2019-02-25

创建、使用、删除数据库

  • create database if not exists 数据库名;
  • use 数据库名;

  • drop database if exists 数据库名;

    添加、删除、清空表

  • create table if not exists 表名（

    字段名 类型 约束，
 ...
 id int not null auto_increment,
 primary key(id)

    ）’

  • 删除表
    drop if exists 表名;

  • 清除表
    truncate 表名；

    约束

  • not null

  • default
  • unique

  • primary key
    主键：每条记录的唯一标识的字段

    修改表

  • 添加字段
    alter table 表名 add column
    字段名 类型约束，
    …;

  • 删除字段
    alter table 表名 drop column 字段名;
