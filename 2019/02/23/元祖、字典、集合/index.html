元祖、字典、集合

发表于 2019-02-23

元祖*（tuple）

不可变、有序的容器型数据类型
元祖不可被修改，元素可以被修改

  • tuple1=（1,2,3）
  • tuple2=1,2,3

    特殊的获取方式

1、变量1，变量2…=元祖 –用变量完全（必须）获取元祖的值
2、*变量 可以获取元祖中剩下的东西（一个新的列表）

  • 可以通过调整位置取相对位置剩下的值
  • 带*的变量只有一个

    排序

sorted(序列):排序并产生新列表
.sort():将原列表排序，所以 元祖.sort()=None

join的使用

字符串.join(序列)

  • 将序列中的字符串取出并用指定的字符串链接

    str(m)

转换字符串的本质是给m加上””

  • 或者’’?

    字典(dict)

可变、无序的容器型数据类型
{key1:value1,key2:value2…}
键（key）：必须是不可变且唯一的（一般是字符串）

  • 如果键可变 会报错
    *如果键不唯一 后面的数据覆盖前面的数据

    查

1.dict[key]
2.dict.get(key)

  • key不存在不会报错，并取一个默认值none
    3.dic.get(key,a)
    *key不存在是，取值a

    增加和修改

dict[key]=a
*添加key:a或者修改key:a

删除

1.del dict[key] 删除key:a
2.dict.pop 删除key:a并返回a

循环与字典

1.for key in dict:
2.for value in dic.values:
3.for key,value in dict.items()s:

  • 本质：

不推荐使用（运行效率低）

运算和方法

1.dict1==dict2(不看顺序，因为dict无序)
不支持>和<
2.in/not in
只支持Key不支持value
3.max(),min()
求Key
4.sum()
求key（数字）的和

转换

1.dict(数据)
*将数据转换为字典，要求数据是具有两个元素的序列
2.将字典转换位其它序列，只取key值。

dict.fromkeys(序列，值)

  • 以序列中所有的元素为key,值为value创建一个新字典
  • dict.keys()
    取出key创建一个序列
  • dict.values()
    取出value创建一个序列

  • dict.items()
    取出key和value创建一个元祖，再将所有元祖创建一个序列

    dict.setfault(key,value=a)

  • 当 key 不存在时 ，添加 key:a

    dict1.update.dict2

  • 使用dict2中的key:value更新（修改或添加）dict1

    集合

可变，无序；元素是唯一并且不可变(暂时不能是列表、字典、集合)
集合可以被修改，元素不能被修改

  • {元素1，元素2，元素3…}

  • m={} 空的dict
    n=set()空的set

  • set(list)可以去掉list中重复的数

查

  • 集合不能单独获取，只能遍历（for n in set）

    增

  • set.add(元素)

  • set.update(序列) 将序列中的元素添加到集合中

    删

  • set.remove(元素) 删除指定元素

    数学集合运算

  • 交集（&）
    两个集合公共的元素产生一个新的集合

  • 并集（|）
    两个集合所有元素产生一个新的集合
  • 差集（-）集合1-集合2
    集合1去掉集合(1&2)产生的新的集合
  • 补集（^）
    去掉公共部分，剩下的所有元素产生一个新的集合
  • 子集和的判断（>,<）
    {1,2,3}>{1,2,3} false
    {1,2,3}>={1,2,3} true
